# TRANSCENDED — Cyberpunk Idle RPG

A cyberpunk-themed idle RPG where you fight enemies, gather resources, and evolve beyond limits.

## Quick Start

```bash
npm install
npm run dev
```

Visit `http://localhost:5173/transcended-idle/`

## Build for Production

```bash
npm run build
```

Output in `dist/` folder — deploy to any static hosting.

## Features

- **Combat System** — Fight procedurally generated enemies
- **Mining & Forestry** — Resource gathering with upgrades
- **Bestiary** — Track kills and unlock bonuses
- **Skills** — Enhance your combat abilities
- **Ascension** — Prestige system with permanent upgrades
- **Daily Challenges** — Rotating special events
- **Offline Progress** — Continue earning while away

## Tech Stack

- Svelte 5
- TypeScript
- Vite

## Deploy

Static files in `dist/` can be hosted on:
- GitHub Pages
- Netlify
- Vercel
- Cloudflare Pages
- Any web server

## Desktop App

To create a desktop version, use:
- **Electron** — `npm install electron`
- **Tauri** — `npm create tauri-app`
- **Neutralinojs** — `npm install @neutralinojs/lib`
